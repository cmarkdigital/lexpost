# LexPost

App de marketing jurídico com IA — gera carrosséis e posts para Instagram usando o Claude.

## Como rodar localmente

```bash
npm install
npm start
```

## Variáveis de ambiente necessárias

Crie um arquivo `.env` na raiz com:

```
REACT_APP_ANTHROPIC_KEY=sk-ant-...
```

## Deploy no Vercel

1. Conecte este repositório no Vercel
2. Adicione a variável `REACT_APP_ANTHROPIC_KEY` nas configurações
3. Deploy automático
